# DishGen

DishGen is a React web application that generates recipes from ingredients you already have at home. Users enter their available ingredients, and the application uses an AI model through the Hugging Face Inference API to create a recipe tailored to those ingredients.

## Features

- AI-powered recipe generation
- Dynamic ingredient management
- Dark mode support
- Responsive user interface
- Tailwind CSS styling
- Secure API key management through environment variables

## Demo

Enter ingredients such as:

- Chicken
- Garlic
- Onion
- Rice

Click **Generate Recipe** to receive a complete recipe generated by the AI.

## Installation

### Clone the repository

```bash
git clone https://github.com/franz2103/dishgen.git
cd dishgen
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env` file in the project root:

```env
VITE_HF_API_KEY=your_huggingface_api_key
```

### Start the development server

```bash
npm run dev
```

## How It Works

1. Users enter available ingredients.
2. The ingredients are combined into a prompt.
3. The prompt is sent to a language model through the Hugging Face API.
4. The AI generates a recipe based on the provided ingredients.
5. The recipe is displayed within the application.
